RIVANOX XML Sitemap Filter for WordPress is a focused **WordPress XML sitemap plugin**, **sitemap filter**, sitemap control tool and technical SEO workflow for site owners, administrators, agencies, freelancers and business websites.

## Frequently asked questions

### What is a WordPress XML sitemap filter?

A WordPress XML sitemap filter controls which website URLs are included in sitemap output. This can include pages, posts, custom post types, taxonomies, users, authors and multilingual entries.

### Can I use the free plugin on a business website?

Yes. The free core plugin may be used for private, professional and business WordPress projects.

### Is this a replacement for a complete WordPress SEO plugin?

No. It is a focused XML sitemap and indexing-readiness tool. It can complement an existing SEO plugin and manual technical SEO review.

### Does the plugin support robots.txt and canonical checks?

Yes. It includes robots.txt guidance and canonical/noindex awareness checks as part of the sitemap review workflow.

### What does the Pro version add?

Pro adds advanced audits, guided fixes, monitoring, verification, rule dry-runs, sitemap snapshots, XML coverage optimization, migration and orphan checks, IndexNow notifications, CSV evidence and agency reports.
